Коран для Windows Phone является свободным, открытым приложением для чтения Корана для телефонов Windows, основанным на проекте Quran Android (http://android.quran.com/). Большинство функций находятся в стадии разработки. Пожалуйста, присылайте мне свое пожелания и не забывайте упоминать всех участников этого проекта в ваших молитвах!

Коран для Windows Phone предоставляет следующие возможности в текущей версии:
- Кристально четкий текст в Мадани формате
- Аудио чтение с подсветкой аятов
- Плавный эффект перехода страниц
- Ландшафтный режим для увеличения шрифтов и более удобного просмотра текста
- Неограниченное количество закладок
- Поиск
- Переводы на различные языки (в настоящее время, арабский тафсир, английский сахих перевод, французский, немецкий, индонезийский, малайский, испанский, турецкий, транслитерация, боснийском, русский, бенгальский, курдский, сомали, голландский, суахили, малаялам, азербайджанский, тамильский и урду)
